Welcome to our short Backbone.js Tutorial!

Today, our goal is to teach you to use Backbone for your simple app in order to create a contact list.

Procedure
❅ Follow Page 2 to download or configure Backbone into your code. For our demonstration purposes, we used Method 2 where we added the code into our HTML file.

❅ Create your standard HTML template and load in the CDNs.
❅ We included Bootstrap for our app in order to have a standard template and to be able to use certain functions.



❅ Create a div with class container for the app
❅ Create your desired columns by defining a table (we used Name, Phone#, E-mail and Action)



❅ Open a new .js file – we will begin to create our Backbone scripts.
❅ Start by creating a Model – a model is where data is presented on the client side.
❅ ContactList is one of our classes that extends Backbone.Model – can provide default values (we did not).



❅ Now we create a Collection – a Collection groups Models together (like a JavaScript array)


❅ The url would be used if we have a database to connect to for storing all of our data, which would require more knowledge outside the scope of our project (such as MongoDB, Mongoose, etc.)

❅ The next thing we will create are Views – views receive information about the data change and changes accordingly.
❅ We will create two views – one will be for each individual contact that we create and the other will be for all of the contacts collectively.



❅ We will want to put the ContactView and ContactsView in a tbody in the HTML. Return to your HTML file and create a <tbody> in your table, with a class of contacts-list, and add <tr>s



❅ Want to now create the views
